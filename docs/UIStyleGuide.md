# UI Style Guide

Goal:

Cozy modern farming game.

Inspired by:

- DoorDash
- Uber Eats
- Stardew Valley
- Animal Crossing
- Steam Deck UI

---

## Design Language

Large cards.

Rounded corners.

Soft spacing.

Clear typography.

Minimal borders.

---

## Spacing

Page Padding

18

Section Gap

8

Card Padding

12

Card Gap

8

Button Height

40

Hero Card

180

## Cards

Title

Main content

One primary button.
